Predictive Sales Audit & Forecasting Dashboard

​Project Overview
​This project transforms historical sales data into a forward-looking analytical tool using Power BI. The goal was to build a dashboard that not only tracks total revenue (currently $20.95M for the selected region) but also anticipates future fluctuations to assist in operational planning.

​Key Technical Solutions
* ​Hierarchy & Drill-Down: Implemented a continuous date hierarchy to transition from yearly overviews to granular monthly trends, ensuring a smooth timeline for the forecasting engine.

* ​Handling Seasonality Errors:Successfully resolved the "Seasonality is out of range" error by adjusting the granularity of data points and aligning the seasonality settings with the available historical range.

•​Predictive Modeling
Applied a 12-month forecast with a 95% Confidence Interval, providing a clear visualization of expected sales performance alongside potential upper and lower bounds.

​Visual Features
* ​Executive Summary: A dynamic text box that summarizes the audit objective and key metrics for stakeholders.

* ​Regional Slicers: Interactive filters for markets like Mexico, France, Germany, and the United States to allow for localized data deep-dives.

* ​High-Contrast Styling: Optimized trend lines (white for historical, black for forecast) to ensure clarity against dark-themed backgrounds.

​Tools Used
* ​Power BI Desktop: Data modeling, DAX measures, and advanced analytics.
